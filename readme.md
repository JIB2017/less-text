# Summize

![Summize](https://github.com/JIB2017/less-text/assets/31837873/6cf8a92c-9609-424a-8ceb-e062c3634e1d)

Summize es tu herramienta si lo que estás buscando es poder resumir esos textos largos en los que lo que realmente importa son solo unas pocas líneas de texto que resumen todo el contenido.

## Como se hizo:

1. **Tecnologías usadas:** HTML, Tailwind CSS, Javascript, React y Redux Toolkit.

2. **UX/UI:** Un hermoso diseño inspirado del sitio (https://dub.sh/) con degradados complejos.

3. **Solicitudes avanzadas con RTK:** Solicitudes a la API "Article Extractor and Summarizer" de RapidAPI con Redux Toolkit.

3. **Guardado del historial:** Se almacena en local todas las solicitudes hechas previamente.

## Lo que aprendí:

Este proyecto lo hice para usar Tailwind CSS y la nueva versión de Redux Toolkit. Ambas son excelentes herramientas que me permitieron agilizar el proyecto y terminarlo en un menor tiempo del esperado.
